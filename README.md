# Extra Colors Mod
An Among Us mod that adds extra colors into the game

# Extra Colors:
- Fortegreen
- Coral
- Olive
- Rose
- Gray
- Tan
- Maroon
- Banana
- Seaform Green
- Medium Purple
- Gold
- Gamboge
- Chocolate
- Dodger Blue
- Violet
- Teal
- Midnight Blue
- Rainbow
